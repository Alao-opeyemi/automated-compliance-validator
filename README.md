# Automated Compliance Validator

## Overview

Python-based AWS compliance validator that checks common cloud security
requirements including public S3 access blocking, encryption at rest, and
access logging. Outputs a JSON compliance report with findings mapped
to specific security standards.

## Standards Alignment

- NIST 800-53 Rev 5: AC-3, AU-2, SC-28
- SOC 2: CC6.1, CC7.2
- ISO 27001: A.10.1, A.12.4
- PCI-DSS: 3.5, 10.2

## Files

- compliance_checker.py - Main compliance scanning script
- .github/workflows/compliance.yml - Automated CI/CD compliance pipeline
- compliance_report_sample.json - Example output showing report structure

## Sample Output

An example JSON report is included to demonstrate the structure of compliance
results generated by this tool.

## Disclaimer

This is a reference validator and not a replacement for a full audit.
